# Фронтенд для AI-ассистента для уроков

Интерактивный интерфейс чата для работы с системой извлечённых уроков.

## Возможности

- 💬 Интерактивный чат-виджет
- 🔍 Поиск извлечённых уроков по запросу
- 📝 Три типа генерации контента:
  - **Вопрос** (по умолчанию) - получить ответ на вопрос
  - **Письмо** - сгенерировать официальное письмо
  - **Документ** - создать документ на основе найденных уроков
- 📚 История просмотренных уроков
- 🎨 Современный и адаптивный дизайн

## Быстрый старт

### 1. Настройка

Убедитесь, что бэкенд запущен на `http://localhost:8000`. Если используете другой адрес, измените настройки в `js/config.js`:

```javascript
const API_CONFIG = {
  baseUrl: 'http://localhost:8000',  // Измените при необходимости
  // ...
};
```

### 2. Запуск

**Рекомендуется:** Откройте `demo.html` для демонстрации полной интеграции:

```bash
# Откройте 
http://localhost:8080/demo.html
```

Или используйте `index.html` для чистого интерфейса чата.

Используйте локальный сервер:

#### Вариант А: С помощью Python

```bash
# Python 3
python -m http.server 8080
```

```bash
# Откройте в браузере
http://localhost:8080
```

#### Вариант Б: С помощью Node.js (http-server)

```bash
# Установите http-server глобально (один раз)
npm install -g http-server

# Запустите сервер
http-server -p 8080
```

```bash
# Откройте в браузере
http://localhost:8080
```

#### Вариант В: С помощью VS Code Live Server

1. Установите расширение "Live Server" в VS Code
2. Откройте `index.html`
3. Нажмите "Go Live" в статус-баре

## Использование

### Основной интерфейс

1. **Открытие чата**: Нажмите на кнопку 💬 в левом нижнем углу
2. **Выбор типа контента**: Выберите один из трёх типов перед отправкой:
   - **Вопрос** - для получения ответа
   - **Письмо** - для генерации письма
   - **Документ** - для создания документа
3. **Отправка запроса**: Введите текст и нажмите Enter или кнопку отправки
4. **Просмотр результатов**: Система отобразит ответ и ссылки на источники

### Расширенный режим

Нажмите кнопку ⤢ для переключения в полноэкранный режим с дополнительными возможностями:

- Более просторный интерфейс
- История просмотренных уроков
- Быстрый доступ к предыдущим запросам

## API Интеграция

Фронтенд взаимодействует с бэкендом через следующие endpoints:

### POST /api/v1/chat

Отправка запроса к ассистенту.

**Request Body:**

```json
{
  "question": "Ваш вопрос",
  "chat_type": "question|letter|document",
  "top_k": 5,
  "use_query_refinement": true
}
```

**Response:**

```json
{
  "answer": "Ответ ассистента",
  "question": "Исходный вопрос",
  "context_chunks": [...],
  "sources_count": 5,
  "chat_type": "question"
}
```

### GET /api/v1/health

Проверка состояния бэкенда.

## Типы контента

### 1. Вопрос (question)

Используется для получения ответа на вопрос на основе извлечённых уроков.

**Пример:**

- "Как избежать срыва сроков?"
- "Какие технологии 5С используются?"

### 2. Письмо (letter)

Генерирует официальное письмо на основе контекста.

**Пример:**

- "Составить письмо о внедрении технологий 5С"
- "Написать обращение по вопросу безопасности"

### 3. Документ (document)

Создаёт структурированный документ.

**Пример:**

- "Создать отчёт о лучших практиках"
- "Подготовить презентацию о бетонировании"

## Кастомизация

### Изменение цветовой схемы

Откройте `css/styles.css` и измените CSS переменные:

```css
:root {
  --primary: #3b6bf2;         /* Основной цвет */
  --primary-light: #4f7df6;   /* Светлый оттенок */
  --bg: #ffffff;              /* Цвет фона */
  --text: #1f2d3d;            /* Цвет текста */
}
```

### Изменение предзаполненных тем (chips)

В `index.html` найдите элементы с классом `.chip` и измените их содержимое:

```html
<div class="chips" id="chipsBar">
  <button class="chip">Ваша тема 1</button>
  <button class="chip">Ваша тема 2</button>
  <!-- ... -->
</div>
```

## Отладка

### Проверка подключения к API

Откройте консоль браузера (F12) и посмотрите логи:

- `API Health: ...` - статус бэкенда при загрузке страницы
- `Chat type changed to: ...` - изменение типа контента
- Любые ошибки API будут отображены в консоли
